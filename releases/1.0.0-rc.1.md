# Nano Markup 1.0.0-rc.1

This is the first release candidate for Nano Markup 1.0. It begins the
compatibility-testing period; it is not the final stable 1.0 specification.

The release contains 112 normative shared cases: 95 decoder cases and 17
writer cases. It also contains seven informative JSON/Nano example pairs.

## Interoperability evidence

On 2026-07-24, the complete decoder, writer, and cross-read protocol passed
with the pre-commit working trees of both independent implementations. Their
immutable evidence commits will be recorded here before the RC tag is created:

Every valid and invalid decoder fixture passed both adapters. Every shared
writer value passed with LF and CRLF, and each implementation decoded the
other implementation's output to the expected tree. Both adapters also parsed
all informative examples.

Known deferred features are listed in `SPEC.md`. Correctness, security, or
interoperability issues discovered during the release-candidate period should
be reported before stable 1.0 is published.
