# Nano Markup specification errata

This file records known errors in released, immutable Nano Markup
specifications. It does not modify the meaning of a tagged release.

An erratum identifies the affected tag and location, describes the problem and
practical impact, and points to the later version that resolves it. Editorial
corrections that do not change normative behavior may be explained here. Any
correction that changes accepted syntax, a decoded tree, an error category, or
conformance behavior requires a new specification version under
`VERSIONING.md`.
